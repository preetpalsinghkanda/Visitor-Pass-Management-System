# VISTRA - Visitor Pass Management System
## visitor pass management system (vpms) is a web application , it's use to manage visitors and enhancing security in organizations, corporate office, schools, hospitals, and residential societies.


# Live Demo 
```https://visitor-pass-management-system-hvtr.vercel.app```


# How to use?

## Visitor Guide
1. open the live web app using above link.
2. Register yourself from signup page.
3. Request visit pass by selecting your host and necessary details.
4. After approval visit there and show QR code at the entrance.
5. Security staff scan the QR code to verify entrance.
6. For exit show again the same QR at the EXIT gate.

## Employee Guide
1. Select employee experience and login yourself.
2. Verify all the requested visits for you.
3. Review the visitor details and approve or reject the request.
4. Once approved, the visitor can generate QR code for visit.

## Security Guide
1. Sign in yourself using authorized ID and Pass.
2. Scan the visitor QR code at the entrance.
3. Verify and Review all the details of visitor.
4. Allow or reject the visitor based on the verification results.
5. Scan the same QR code again at the exit.
